StandardTrailer

The standard FIX message trailer

Tag Name Req'd Description
93 SignatureLength N Number of bytes in signature field
89 Signature N Electronic signature
10 CheckSum Y Three byte, simple checksum (see Volume 2: "Checksum Calculation" for description). ALWAYS LAST FIELD IN MESSAGE; i.e. serves, with the trailing <SOH>, as the end-of-message delimiter. Always defined as three characters. (Always unencrypted)
